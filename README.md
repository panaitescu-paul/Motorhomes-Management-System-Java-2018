# Motorhomes Management System - Java - 2018

### Features:
- Login access to 4 types of users
- Create, Read, Update, Delete for Motorhomes
- Create Rentals

### Technologies:
- Java
- MySQL
- Spring & Tymeleaf
- HTML, CSS, JS
